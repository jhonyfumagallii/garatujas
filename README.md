## HTML para eu do futuro!

  Html significa hyper text markup language
  Para colocar os titulos, usa h1 h2 h3 h4 h5 h6
  todas as coisas devem começar e terminar, ex: <h1>titulo<h1/>
  <strong> é negrito
  lang indica a linguagem, ex <html lang"pt-BR">
  <meta> não há fechamento, vale pro código todo, NÃO MUDA NADA NO SITE
  charset = conjunto de caracteres, ex: utf-8, que contem todos os caracteres de todas as linguas, incluindo emojis
